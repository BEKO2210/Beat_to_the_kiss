## Anpassungen

### Social Media Links ändern
In der `index.html` Datei, Section "Contact":
```html
<a href="DEIN_INSTAGRAM_LINK" target="_blank" class="social-link">
<a href="DEIN_YOUTUBE_LINK" target="_blank" class="social-link">
<a href="DEIN_SOUNDCLOUD_LINK" target="_blank" class="social-link">
```

### Farben ändern
Im CSS-Bereich (Style-Tag) kannst du:
- `#ff4444` (rot) durch deine Wunschfarbe ersetzen
- Background-Farben anpassen

### Neue Alben hinzufügen
Kopiere ein `.album-card` Element und passe Titel, Jahr und Typ an.
